DNS Record Types
| Record        | Description |
| ------------- | ------------- |
| A             | IPv4 address  |
| AAAA          | IPv6 address  |
| CNAME         | Canonical name record  |
| MX            | Mail record  |
| NS            | Nameserver record  |
| PTR           | Pointer record  |
| SOA           | Start of authority record  |
| SRV           | Service location record  |
| TXT           | Text record  |
